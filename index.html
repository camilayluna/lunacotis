¿Qué es CSS?

CSS es un lenguaje que se utiliza para estilizar los elementos de un lenguaje de marcado como HTML.

¿Que es html?

HTML sirve exactamente para hacer de tu página web un proyecto con estilo propio, podrás colocar imágenes, textos, videos y todo lo que necesites mediante sus etiquetas .

¿Que es javascript?

JavaScript es el único lenguaje de programación que funciona en los navegadores de forma nativa (lenguaje interpretado sin necesidad de compilación). Por tanto se utiliza como complemento de HTML y CSS para crear páginas webs. .
